- i == 1, j==3, k == 4
- i == 2, j==3, k == 4

- よって10回


## 公式より

```
nCr = n!/(r! * (n-r)!)

nC3 = n!/(3*2*1 * (n-3)!)
    = n!/(6 * (n-3)!)
    = {n * (n-1) * (n-2) * (n-3)!} / {6 * (n-3)!}
    = 1/6 * n (n-1) * (n-2)
```    

よって、これを展開して最大次数を考慮すると `O(N^3)` となる。
