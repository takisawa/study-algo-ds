# 2.1

## T1(N) = 1000N

十分大きな整数Nに対して `1000N/N <= 1000` となることから `O(N)` となる。


## T2(N) = 5N^2 + 10N + 7

`(5N^2 + 10N + 7)/N^2` => `5 + 10/N + 7/N2`

十分に大きな整数Nに対して `10/N + 7/N2` <= 1 となるので、
`(5N^2 + 10N + 7)/N^2 <= 6` が成り立つ。

よって `O(N^2)` となる。


## T3(N) = 4N^2 + 3N√N

- N√N は N^2 で抑えられる（2.1.pngを参照）
- よって `O(N^2)` となる


## T4(N) = N√N + 5N logN

- N√Nは N logN で抑えられる（2.1.pngを参照）
- よって `O(logN)` となる


## T5(N) = 2^N + N^2019

- N^2019 は 2^N によって抑えられる
- 境界値は N=30,030
- よって `O(N^2)` となる
