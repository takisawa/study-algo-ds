## 問題

```
  2 a
* b c
-----
d 3 e
f g
-----
h 4 i
```


## メモ

```
  2 7
* 3 5
-----
1 3 5
8 1
-----
9 4 5
```

## 回答1

```
g=1
  => 4-g=3より
b=3,a=7
  => gを1にするには、a=1,b=1 / a=3,b=7 / a=7,b=3 / a=9,b=9 のどれか
  => a=1,b=1の場合、dの横の3に至ることができない
  => b=7, b=9 の場合、fg行の桁が溢れてしまう
  => よって、a=7,b=3のみ残る
c=5
  => dは0になれないので、cは5以上
  => dの横が3なので、c=5に決まる
e=5,i=5,d=1,h=9に決まる
```
